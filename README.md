# About Project

In my SmartInsure: ML-Based Insurance Premium Prediction System, I used data visualization extensively across the full machine learning lifecycle. During exploratory data analysis, I created distribution plots, correlation heatmaps, and boxplots using Matplotlib, Seaborn, and Plotly to understand relationships between features such as BMI, age, and insurance charges, as well as to identify skewness and outliers.

I also leveraged visualization for model evaluation and interpretability. I analyzed confidence intervals and prediction intervals to assess uncertainty in predictions, and used SHAP-based visualizations to interpret feature importance and understand how individual variables influenced premium estimates. For modeling, I built a LightGBM regression model, optimized using Optuna with K-fold cross-validation, and tracked experiments using MLflow to compare performance across runs and ensure reproducibility.

Finally, I deployed the model as a REST API using FastAPI and built an interactive Streamlit application to enable real-time predictions and visualize outputs for end users. Overall, this experience helped me use visualization not only for data exploration, but also for improving model performance and effectively communicating insights.
